What is this?

This is a very basic and simple static website that I made to present my tutorial (brazilian portuguese) on JUDCon 2014 Brazil.

I developed this site using some awesome and open source tools made by JBoss guys:

  1. Awestruct. A framework for creating static HTML sites, inspired by the Jekyll. By using this tool, I’m now creating my new website and others, like this.

  2. Asciidoctor. A fast text processor and publishing toolchain for converting AsciiDoc to HTML5, DocBook and more. Particulary, I’m very grateful to Dan Allen for his work on this tool because I learned many things as a result of its use. I’m using Asciidoctor to write my book about JBoss EAP (Brazilian Portuguese).

They were also integrated with some other cool tools/ technologies:

  1. Bootstrap

  2. Slim

If you are interested in know how to produce your static website using this tools, check out a tutorial that I presented this year on JoinCommunity 2014 and, also, read this page.
